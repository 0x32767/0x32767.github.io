EoSDecomp

Overview

EoSDecomp is a decompilation effort that aims to create a byte accurate decompilation of TouHou 6 ~ The Embodiment of Scarlet Devil. The game was originally made with DirectX8 which was become more and more unstable with newer windows updates, causing unexpected crashes and unplayability. Once the source code has been extracted, the plan is to migrate the game to a newer version of Direct X.

Current state

The project is available on github and has a discord server.

What I learned with the project
